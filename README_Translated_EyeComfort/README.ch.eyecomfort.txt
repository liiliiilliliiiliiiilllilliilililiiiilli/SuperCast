SuperCast——全新级别的推特体验。

音频托管，让您可以收听、创作和讨论播客。
没有广告、用户协议和数据收集——互联网就该如此。

核心理念：

    迄今为止，“音频托管”的概念尚未明确，分散在本质上类似的平台中。
    之前没有像 YouTube (https://www.youtube.com) 那样原创的音频服务，它提供视频观看功能；也没有像 Pinterest (https://www.pinterest.com) 那样用于制作拼贴画；也没有像 Telegram (https://telegram.org) 那样提供免费交流的音频服务。
    播客和录音没有独立的权利，在没有得到适当发展的情况下只能作为次要的存在。

    此外，现有的音乐服务，例如 Yandex.Music (https://music.yandex.ru) 或 Spotify (https://open.spotify.com)，尽管缺乏替代方案，却通过强迫用户付费订阅并侵犯其私人数字空间（例如，通过加密方式禁止访问已下载到设备上的音频文件，剥夺用户对其软件的信任）破坏了用户体验。

    SuperCast 是一个注重软件自由和用户行动自由的音频平台。
    在这里，播客、录音、出版物附件以及任何内容都可以不受限制地下载、重复使用和分发——并且没有任何审查。
    没有人限制您的外部活动，也不会侵犯您的个人活动——这些是该服务的原则。

服务机制：

    出版物:

        出版物（或帖子）是作者创作内容的主要单位。
        帖子位于用户个人资料中，可以包含文本、附件和音频媒体（播客/录音）。
        任何出版物都可以进行评分，并以文字或语音进行评论，或在您的个人资料中公开回复。

    播客和录音:

        播客是一种重要的视听格式。
        您可以收听播客，并同时观看对话的实时记录。
        播放器具有多种视觉显示类型，播放过程中，会以注释和附件的形式弹出指定特定时间码的实时提示。
        播客可以下载并在后台收听。
        您可以通过内置编辑器或导入现成的预设，在该服务的任何应用程序中手动创建播客。

        录音是作为评论或出版物附件的简短语音消息，无法预先录制。
        任何录音都可以转录为文本。
        一个录音也可以作为语音状态固定在个人资料中。

    搜索引擎:

        这看似一个简单的功能，但我们内置了几乎无处不在的搜索功能，并带有筛选器——它可以在任何屏幕上使用，并帮助您搜索音频媒体中的用户、出版物、评论或有趣时刻。

    主题订阅源和订阅:

        为了清晰起见，作者可以将其个人资料中的出版物按主题分发到不同的出版物订阅源中，并为它们添加名称和描述。
        听众可以分别订阅每个订阅源（或一次订阅所有作者），而作者的订阅者数量由订阅了至少一个订阅源的用户数量组成。

        听众可以将多个订阅合并到单独的“查看订阅源”中，并分别为每个订阅源设置内容显示参数。
        “查看订阅源”可从主屏幕访问，帮助您更好地浏览订阅内容。

    存储和下载:

        任何内容都可以标记并添加到“已保存”或下载。
        此类内容存储在“存储”中，收听历史记录也保存在此处。

        存储是一个树状结构，包含当前设备上下载内容和虚拟保存内容的文件夹和文件，可在多台设备之间访问。保存的内容可以根据您的喜好进行分组，系统会自动按类型进行划分。

特点：

    我们不强迫用户观看或收听广告，不进行付费订阅，也不设置任何年龄限制。
    我们的工作不涉及与各国政府合作，也不会根据政治或宗教观点对用户进行审查。
    本服务的唯一审查对象是普遍接受的事物。

分发和使用：

    本服务正在开发中——Android 版应用程序正在准备中。
    之后，我们将发布网站、iOS 版应用程序以及 Linux、Windows 和 MacOS 版桌面版。

    该网络免费，并将在所有地区开放。
    我们将添加慈善订阅，它不会带来任何额外功能，但可以维持服务的正常运行。

    该项目拥有开放的代码库，但目前不接受第三方编辑。

此外：

    Facebook (https://facebook.com)、VK (https://vk.com)、Twitter (https://x.com) 和其他革命性“垃圾”平台的经验表明，社交网络对人们的生活和体验承担了过多的责任。
    几乎所有这些服务迟早都会陷入金钱、政治和欺骗的泥潭——我们不希望我们的服务遭遇同样的命运。
    我们坚信SuperCast会成功，但我们也理解它与其他项目一样脆弱。
    从其他社交网络的创始人口中听到这样的话，实属罕见——不是吗？

    正因如此，我们挑战所有其他服务，捍卫用户自由的理念，并希望树立一个榜样，让人们知道自由互联网是值得奋斗的。
    我们希望这个榜样能够引领我们走向真正想要的，或者至少阐明我们真正需要的是什么。
    我们希望最终结果能够如其所愿。
